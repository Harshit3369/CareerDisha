# CareerDisha: Narrative Journey Mapping - Product Requirements Document

## Executive Summary

CareerDisha transforms traditional college discovery from static ranked tables into a dynamic "Narrative Journey Map." Users input their qualitative life situation in natural language, and the system generates a branching story map with four distinct paths representing different risk/reward profiles.

---

## 1. The Narrative Engine (Input Logic)

### 1.1 NLP Input Processing

**User Input Example:**
> "I'm from a small town in Bihar, my parents want me to stay close, I scored 78% in PCM, I want to do something with computers but I'm not sure if engineering is right for me."

**Life Archetype Classification:**

| Archetype | Key Indicators | Primary Path |
|-----------|---------------|--------------|
| **The Anchored Explorer** | Family constraints, location-specific, uncertainty | Path A (Safe Harbor) |
| **The Calculated Climber** | Moderate scores, regional mobility, upward ambition | Path B (Moderate Reach) |
| **The High-Stakes Dreamer** | High scores, willing to relocate, ambitious | Path C (High-Impact Leap) |
| **The Global Seeker** | High scores, international aspirations, financial capacity | Path D (Global Horizon) |

### 1.2 NLP Parsing Schema

```typescript
interface LifeProfile {
  location: {
    state: string;
    tier: 'metro' | 'tier2' | 'tier3' | 'rural';
    familyConstraint: boolean;
  };
  academic: {
    score10: number;
    score12: number;
    stream: 'PCM' | 'PCB' | 'Commerce' | 'Arts' | 'Vocational';
    percentile?: number; // JEE/CUET etc.
  };
  psychological: {
    certainty: 'exploring' | 'decided' | 'undecided';
    riskTolerance: 'low' | 'moderate' | 'high';
    interests: string[];
  };
  constraints: {
    budget: 'low' | 'moderate' | 'high';
    relocation: boolean;
    visaRequired: boolean;
  };
}
```

### 1.3 Archetype Detection Logic

```
IF familyConstraint = true AND budget = low → Anchored Explorer
ELSE IF riskTolerance = 'moderate' AND relocation = true → Calculated Climber
ELSE IF riskTolerance = 'high' AND score12 >= 85 → High-Stakes Dreamer
ELSE IF visaRequired = true AND budget = 'high' → Global Seeker
```

## 2. The Branching Story Map (Visual UX)

### 2.1 Core Visual Architecture

The map uses a horizontal branching timeline with the user at the center. Each path is a vertical "story lane" that users can expand/collapse.

```
[You Are Here] → [Path A] → [Path B] → [Path C] → [Path D]
                Safe      Moderate  High-Impact  Global
                Harbor    Reach     Leap         Horizon
```

### 2.2 Path Definitions

#### Path A: The Safe Harbor
- **Target User:** Anchored Explorers
- **Location:** Within 100km radius
- **Options:** BSc CS, BCA, Local State Universities
- **Color:** Green (#22C55E) - Safety/Stability
- **Badge:** "Low Pressure • Proximity Priority"

#### Path B: The Moderate Reach
- **Target User:** Calculated Climbers
- **Location:** Regional cities (Patna, Ranchi, Lucknow, etc.)
- **Options:** Private universities, State universities with good CS programs
- **Color:** Blue (#3B82F6) - Trust/Progression
- **Badge:** "Upward Mobility • Manageable Investment"

#### Path C: The High-Impact Leap
- **Target User:** High-Stakes Dreamers
- **Location:** Major metros, National Institutes
- **Options:** NITs, IIITs, Top Private (VIT, SRM, BITS)
- **Color:** Purple (#8B5CF6) - Ambition/Premium
- **Badge:** "High ROI • Competitive Entry"

#### Path D: The Global Horizon (Premium Only)
- **Target User:** Global Seekers
- **Location:** USA, Canada, UK, Germany, Australia
- **Options:** Community colleges → Transfer, Bachelors abroad
- **Color:** Gold (#F59E0B) - Premium/Worldwide
- **Badge:** "Global Career • Premium Tier"

## 3. The Reality-Check Data Layer

### 3.1 Four-Year Life Projection

Each path displays a collapsible "Life Timeline" showing:

```
Year 1: Foundation
- Placement Rate (After 4 years): 78%
- Median Salary: ₹4.2 LPA
- Hostel Culture: 80% of students, mixed gender wings
- Typical Recruiters: TCS, Infosys, Wipro

Year 2-4: Growth
- Internship Opportunities: 120+ companies visit
- Higher Education Pathways: 25% pursue MCA/Higher studies
- Alumni Network Strength: Moderate (2,000+ active)
```

### 3.2 Student Sentiment Engine

**Data Sources:** Reddit (r/BTechAspirants, r/IndianCollegeReview), Quora, College-specific Facebook groups

**Sentiment Categories:**
- **Academic Rigor:** "Manageable" | "Intense" | "Brutal"
- **Peer Quality:** "Motivating" | "Average" | "Complacent"
- **Faculty:** "Supportive" | "Okay" | "Distant"
- **Campus Life:** "Vibrant" | "Basic" | "Boring"

**Presentation:** 5-star rating with 2-3 sentiment quotes per category

### 3.3 "Life at [College]" Snapshot

Interactive accordion with:
- **Hostel Reality:** Real photos, mess food ratings, WiFi quality
- **Local Commute:** Bus/train availability, city safety index
- **Money Matters:** Part-time job options, monthly expenses
- **Hidden Gems:** Professor mentorship programs, hackathons, clubs

## 4. Emotional Gamification & The Tracker

### 4.1 Path Commitment System

**Replace "Add to List" → "I want this path"**

When user clicks:
1. **Animation:** Path lane glows with particle effects
2. **Haptic Feedback:** Vibration on mobile
3. **Commitment Message:** "You're chasing Path B. Let's make it real."

### 4.2 Path Tracker Logic

```typescript
interface PathTracker {
  selectedPath: 'A' | 'B' | 'C' | 'D';
  requiredExams: {
    examName: string;
    deadline: Date;
    daysLeft: number;
    priority: 'high' | 'medium' | 'low';
    preparationTime: string; // "2-3 months recommended"
  }[];
  milestones: {
    task: string;
    dueDate: Date;
    completed: boolean;
  }[];
}
```

**Auto-population Rules:**
- Path A: BSc CS entrance (state level), CUET (optional)
- Path B: State engineering entrance, Private university exams
- Path C: JEE Main, JEE Advanced, State CETs, Private exams
- Path D: SAT, IELTS/TOEFL, Passport application

### 4.3 Urgency UI Design

**Tracker Header:**
```
[Path B Active] You're 3 exams away from Regional Colleges
┌─────────────────────────────────────────────┐
│ NEXT DEADLINE                               │
│ State Engineering CET                      │
│ Application closes in 47 days • Moderate   │
└─────────────────────────────────────────────┘
```

**Progress Visualization:**
- Progress bar with 4 segments (Exam Prep, Application, Result, Admission)
- Each completed exam fills a segment
- Milestone badges unlock (e.g., "Prep Warrior" after 30 days of study)

## 5. Monetization Framework

### 5.1 Tier Structure

| Feature | Free Tier | Premium Tier (CareerDisha+) |
|---------|-----------|---------------------------|
| Paths Visible | A, B | A, B, C, D |
| Path Saves | 1 | Unlimited |
| Exam Alerts | Manual refresh | Auto reminders (push/email) |
| Global Horizon | Locked | Unlocked |
| Reality Check Data | Limited | Full access |
| Price | ₹0 | ₹999/year or ₹99/month |

### 5.2 Conversion Funnel

1. **Hook:** User sees Path A and B results (free)
2. **Tease:** Path C shows "Premium Preview" with first 3 colleges blurred
3. **CTA:** "Unlock your High-Impact Leap and beyond"
4. **Value:** Show comparison - Free = 1 path, Premium = 4 paths + automated tracking

### 5.3 Upgrade Triggers

- After 1 path saved (showing Path D would be beneficial)
- During exam deadline countdown ("Get alerts for all 3 exams?")
- On result day ("Struggling with choices? Let's explore all 4 paths")

## 6. Technical Implementation Requirements

### 6.1 API Endpoints

```
POST /api/narrative-input
  - Input: natural language prompt
  - Output: LifeProfile + Archetype

POST /api/generate-paths
  - Input: LifeProfile
  - Output: { pathA, pathB, pathC?, pathD? }

POST /api/path-tracker
  - Input: selected path + user profile
  - Output: PathTracker object with deadlines
```

### 6.2 Data Sources Required

- **Entrance Exam Dates:** Yearly scrape from official websites
- **College Data:** NIRF rankings, placement stats, fee structure
- **Sentiment Data:** Reddit API integration, web scraping pipelines
- **Visa/Cost Data:** Education consultants, government portals

### 6.3 UI Component Structure

```
<NarrativeJourneyMap>
  <PathLane path="A" archetype="anchored" />
  <PathLane path="B" archetype="calculated" />
  <PathLane path="C" archetype="high-stakes" locked={!isPremium} />
  <PathLane path="D" archetype="global" premiumOnly />
  <PathTracker selectedPath={activePath} />
</NarrativeJourneyMap>
```
